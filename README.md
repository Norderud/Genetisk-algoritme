# Genetisk-algoritme

Et program som bruker en genetisk algoritme til å lære seg å komme seg frem til et mål

## Forutsetninger

- **Java 20**
- **Maven**

## Kompilere og starte

1. Gå inn i katalogen `genetisk`:
   ```bash
   cd genetisk
   ```
2. Kompiler prosjektet og last ned avhengigheter:
   ```bash
   mvn clean package
   ```
3. Start JavaFX-applikasjonen:
   ```bash
   mvn javafx:run
   ```
